# Data-Professional-Survey-Analysis
Power BI analysis of data professionals survey dataset, exploring trends in roles, salary, programming languages, and satisfaction.

# Objectives
- To understand salary accross the different Job Roles
- Identify favourite Programming Language across Data Professionals.
- Analyse Job Satisfaction both in Salary and Work/Life Balance.
- Explore Difficulty Level in Breaking into Data as a career.
- Explore different countries Survey.

# Tools Used
- Power BI
- Data cleaning and Transformation (Power Query)
- DAX
  
# Key Insights
- Data Scientists and Data Engineers tend to earn higher salaries
- Python is the most widely used Programming Language of Data Professionals.
- Many respondents reported moderate difficulty in breaking into Data.
- Less than Average respondents reported unsatisfaction with their salary
- More than Average respondents reported satisfaction with Work/Life balance.
